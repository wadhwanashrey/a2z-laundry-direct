
# A2Z Laundry – Single Page Web App

## Hero Section
- Bold headline: "A2Z Laundry – Flawless Quality. Transparent Pricing."
- Subtext about real service, no fluff
- Prominent green "Book via WhatsApp" button linking to `https://wa.me/9109920602363?text=Hi%20A2Z%20Laundry%2C%20I%27d%20like%20to%20place%20an%20order.`
- Clean, minimal design with brand colors

## Pricing Grid
- Clear table with per-piece/per-kg rates for each service
- **Minimum order value displayed prominently** next to each rate (e.g., "₹199 minimum")
- No hidden fees section – everything visible upfront

## Services (Core Only)
- Wash & Fold
- Standard Ironing
- Dry Cleaning
- Each with brief, honest description – no fluff

## Turnaround Times
- Two-column layout: Standard vs Express
- Standard: based on max load (e.g., 48-72 hours)
- Express: exact premium cost clearly stated (e.g., "+50% for 24-hour delivery")
- Disclaimer: "Times reflect peak operational load, not best-case scenarios"

## Quality Control Process
- Visual step-by-step breakdown (numbered stages)
- Stages: Inspection → Sorting → Wash/Clean → Post-wash check → Pressing → Final QC → Packaging → Dispatch
- Each stage with a brief, concrete description

## Footer
- Full address, operating hours (8 AM – 10:30 PM)
- WhatsApp contact button (repeated CTA)
- Instagram link
- Simple, no-nonsense layout

## Design
- Clean white background, dark text, accent color for CTAs
- Fast-loading, no heavy images or animations
- Mobile-first responsive design
- Smooth scroll navigation between sections
